  In this section I learned more about network layouts like Bus, Star, Mesh, Hybrid, and ring and how devices are connected on each layout by reviewing their topology diagrams. I also learned about routers, hubs, switches, and WAP's. I learned how network engineers need to understand how all these things are connected with physical and logical layouts... Very interesting!!

Terms to Know:

- Access Layer: The part of the network where end devices connect, focusing on connectivity and port security.

- Bus Topology: A legacy network design where all devices share a single central cable called a backbone, making it simple but prone to total failure if the main cable breaks.

- Core Layer: The high-speed backbone that switches and routes traffic between distribution devices.

- Distribution Layer: The layer that aggregates traffic from access switches and applies policies before sending it to the core.

- Firewall: A network security device that filters traffic between trusted and untrusted networks based on rules.

- Hybrid Topology: A combination of two or more different network topologies, designed to balance performance, scalability, and flexibility.

- Load Balancer: A device that distributes network traffic across multiple servers to prevent overload and maintain performance.

- Logical Layout (or Topology): The path data takes through a network, showing how devices communicate regardless of physical layout.

- Mesh Topology: A network layout where devices are interconnected with multiple paths, providing high reliability and redundancy at the cost of complexity and expense.

- Patch Panel: An organizational board of ports that connects building cables to a switch using short patch cables.

- Physical Layout (or Topology): The physical arrangement of devices and cables in a network.

- Redundancy: The practice of adding duplicate components so that the network can continue running if one part fails.

- Ring Topology: A legacy network layout where devices form a closed loop, passing data in one direction around the circle; efficient but vulnerable to single points of failure.

- Router: A device that connects different networks and uses IP addresses to direct traffic.

- Scalability: The ability of a network to grow and handle more demand without a complete redesign.

- Star Topology: A network layout where all devices connect to a central switch or hub, making it easy to manage and troubleshoot, but dependent on that central device.

- Wireless Access Point (WAP): A device that allows wireless devices to connect to a wired network.
